# CSE499_FlowForge-Smart-Task-Productivity-System

Many students and small teams struggle to stay organized across multiple assignments, deadlines, and responsibilities. FlowForge aims to solve this by creating a smart productivity system that not only tracks tasks, but also:  Helps prioritize work Tracks workload over time Provides simple insights into productivity habits
